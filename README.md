# corezoid-jwt
simple JWT sign for corezoid.


# Example usage
![Git call preview](https://i.imgur.com/fkMPm1e.png)
```javascript
const jwtSign = require("corezoid-jwt");
data.token = jwtSign({your: "data"}, "secret");
```
